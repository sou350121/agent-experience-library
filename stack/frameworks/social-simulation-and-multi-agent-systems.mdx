# 社会模拟与多智能体预测系统 (Social Simulation & MAS)

當智能體從單體助手 (Single Agent) 進化為群體社會 (Multi-Agent Society) 時，我們進入了「數字雙生推演」的新時代。

## 🧠 核心洞见
> **預測未來的最好方式，是在數字世界中把它「跑」一遍。**
> MiroFish 等項目的價值在於：它不是在計算數學概率，而是在模擬人類行為。當我們能大規模模擬 Agent 的社會交互時，我們就在構建一個具備因果推演能力的平行世界。

## 🛠 MiroFish 的架構範式
1. **種子記憶 (Seed Memory)**：通過 GraphRAG 將靜態報告轉化為動態知識。
2. **人格具象化 (Persona Embodiment)**：為數千個 Agent 注入異質的人格背景與行為邏輯。
3. **時序演化 (Temporal Evolution)**：模擬 Agent 之間的點讚、發帖、對話等社交行為。
4. **決策反饋 (Decision Loop)**：上帝視角的變量注入與報告生成。

## 🚀 實踐意義
- **風險對沖**：公關危機或政策出台前的「演習場」。
- **長尾洞察**：挖掘傳統統計學難以捕捉的社會情緒「蝴蝶效應」。
- **Agent 進化**：在群體環境中訓練 Agent 處理複雜社交與衝突的能力。
