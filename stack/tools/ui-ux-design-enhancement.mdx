# UI-UX-Pro-Max：告别“AI 味”十足的界面

> 既然我们已经进入了 Vibe Coding 时代，那么“审美”就不再仅仅是设计师的专利，而是 Agent 的一项**核心插件**。`ui-ux-pro-max-skill` 解决了 AI 生成代码最头疼的问题：**看起来都一样，且缺乏交互灵魂。**

---

## 1. 为什么你需要它？

当你让 Cursor 或 Claude 写一个“漂亮的登录页面”时，它们通常会给你一个标准的 Tailwind 灰白配色 + 方形按钮。虽然能用，但缺乏**高级感**。

`ui-ux-pro-max-skill` 通过注入 **57 种 UI 风格**、**95 套调色方案** 和 **98 条 UX 规范**，强制 AI 遵循更高阶的设计逻辑：
- **拒绝通用性**：不再是默认的灰色，而是带有品牌感的深色调或毛玻璃效果。
- **动效优先**：自动补全 Framer Motion 或 GSAP 的交互逻辑。
- **辅助功能**：自动考虑 contrast ratio（对比度）和 aria-labels。

---

## 2. 安装与配置

该工具是一个 CLI，可以一键为你的 Agent（Cursor, Claude Code 等）注入“设计灵魂”。

### 2.1 全局安装
```bash
npm install -g uipro-cli
```

### 2.2 为你的项目初始化
进入你的项目根目录，运行：
```bash
uipro init --ai cursor      # 如果你主要用 Cursor
uipro init --ai claude      # 如果你主要用 Claude Code
```

这会在你的 `.cursor/rules` 或相关配置中添加一套**设计规则集**。

---

## 3. 核心应用场景

### 3.1 像素级还原 (Pixel-Perfect)
当你截取一张精美 UI 的图喂给 Agent 时，配合该插件，它能更精准地识别出：
- **负空间 (Negative Space)** 的韵律感。
- **阴影层级 (Elevation)** 的细微差别。
- **排版 (Typography)** 的字重对比。

### 3.2 风格迁移
你可以对 Agent 说：
> "用 `ui-ux-pro-max` 的 `Cyberpunk` 风格重构目前的 Dashboard，并使用 `Tailwind` + `Framer Motion` 实现进入动画。"

## 4. 配合 AGENT_CONSTITUTION 使用

在我们的 [Agent 宪法](../../AGENT_CONSTITUTION_UNIVERSAL.md) 中，**“确定性治理”**是核心。而 UI/UX 的一致性就是视觉上的确定性。

**建议指令：**
> "在生成任何 UI 组件前，请先调用 `ui-ux-pro-max` 的 `UX Checklist` 检查该组件的交互是否符合最佳实践（如：加载状态、错误提示、按钮点击反馈）。"
