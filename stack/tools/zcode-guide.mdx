# Z Code：统一 Agent 协作编辑器指南

> Z Code 是由智谱 AI 推出的轻量级 AI 代码编辑器，旨在将强大的 CLI Agent 能力（如 Claude Code, Codex）转化为直观的可视化工作流。

## 1. 核心优势

- **统一可视化界面**：将原本枯燥的命令行 Agent 交互图形化，提供文件管理器、Git 提交面板和内置浏览器。
- **多 Agent 调度**：支持一个 API Key 接入并切换 Claude Code、Codex 和 Gemini，实现智力的按需分配。
- **安全保障机制**：提供操作安全确认、代码审查（Review）和文件版本管理，防止 Agent 误操作导致的代码丢失。
- **内置 MCP 支持**：原生支持模型上下文协议（MCP），可轻松扩展 Agent 的外部工具调用能力。

## 2. 主要功能清单

### 2.1 交互与思考
- **思考模式 (Thinking Mode)**：可视化展示 Agent 的推理过程。
- **编辑历史对话**：支持回溯并修改之前的指令，重新生成结果。
- **文件引入**：通过可视化方式快速为 Agent 提供上下文。

### 2.2 工程化支持
- **版本管理**：安全的文件快照，支持一键回滚。
- **Git 集成**：内置 Git 提交面板，方便 Agent 完成任务后直接提交 PR。
- **命令行面板**：在编辑器内直接执行终端指令。

### 2.3 扩展能力
- **网页浏览器**：允许 Agent 实时访问网页获取文档或调试。
- **多权限模式**：控制 Agent 的写文件、执行命令等权限级别。

## 3. 与现有工具对比

| 特性 | Z Code | CC Switch | Cursor / VS Code |
| :--- | :--- | :--- | :--- |
| **形态** | 轻量级编辑器 | 配置切换工具 | 全功能 IDE |
| **Agent 支持** | 多 Agent 统一 UI | 仅限 CLI 切换 | 插件化支持 |
| **上手门槛** | 极低（图形化） | 中（需配合终端） | 低（插件式） |
| **核心卖点** | **编排层可视化** | 纯净 CLI 环境 | 生态丰富 |

## 4. 总结

Z Code 是目前 **“一人 CEO”** 范式的最佳硬件选型之一。它通过降低工具门槛，让开发者能将更多精力放在 **Taste（方向判断）** 上，而非纠结于命令行的语法细节。
