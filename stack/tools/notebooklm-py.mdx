# NotebookLM-py: CLI-Based Knowledge Engine

`notebooklm-py` 是一個將 Google NotebookLM 功能封裝為命令行工具和 Python SDK 的開源項目。

## 🧠 核心洞见
> **CLI 是 Agent 的 GUI。**
> 對於人類，網頁版提供了精美的 UI；對於 Agent，CLI 和 API 才是它們觸達最強 RAG 能力的真正入口。

## 🛠 安装与配置
```bash
pip install "notebooklm-py[browser]"
playwright install chromium
notebooklm login
```

## 🚀 三种调用范式
1. **Skill 模式**：集成至 Claude Code，實現 `notebooklm generate audio` 的自然語言調用。
2. **CLI 模式**：適合批量處理任務，如 `notebooklm source add <URL>`。
3. **API 模式**：構建自定義 Agent 流程的核心組件。

## 💡 最佳实践
- **异步处理**：在 Python API 中利用 `asyncio` 併發處理多個 Notebook 的生成任務。
- **研究自动化**：將 YTB 技術視頻自動轉錄並生成 Markdown 總結，存入個人經驗庫。
