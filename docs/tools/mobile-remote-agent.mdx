# 工具工作流：手机远程遥控 Agent (智力套利)

> **核心目标**：实现“随时随地”驱动顶尖 Agent 干活，最大化固定订阅费的产出价值。

## 1. 概念：什么是“智力套利”？
由潘致雄 (@nake13) 提出。通过支付固定的月费（如 $20/mo 或 $200/mo），在模型运行成本远超订阅费的情况下，通过高频、高效的调用，实现对 AI 公司计算资源的“反向收割”。

## 2. 硬件与软件准备
- **宿主端 (Host)**：一台性能尚可且网络稳定的 PC/Mac，需保持开启。
- **Agent 端**：安装并配置好 CLI 类型的 Agent，如 **Claude Code**, **Aider** 或 **Cursor (via terminal usage)**。
- **移动端 (Mobile)**：手机或 iPad，安装 **Termius**, **Blink Shell** 或 **Prompt** 等 SSH 客户端。
- **内网穿透/远程访问**：Tailscale (推荐), Cloudflare Tunnel 或公网 IP。

## 3. 设置步骤

### 第一步：宿主端开启 SSH 服务
确保你的电脑可以接受远程连接。

### 第二步：配置 Agent 自动运行环境
建议在宿主端使用 `tmux` 或 `screen`，防止手机网络抖动导致进程中断。
```bash
# 创建一个名为 agent 的会话
tmux new -s agent
# 在其中启动你的 Agent
claude-code
```

### 第三步：手机远程连接
1. 打开 Termius，输入宿主端的 SSH 地址。
2. 连接成功后，输入 `tmux attach -t agent`。
3. 现在，你可以在手机键盘上输入指令，让 Agent 在家里帮你写代码、跑测试、抓数据。

## 4. 适用场景
- **突发灵感**：在咖啡馆想到一个 Bug 修复方案，直接手机指令 Agent 完成修复并推送到 GitHub。
- **长时任务监控**：在路上查看 Agent 重构大型项目的进度。
- **极致省钱**：既然买了 $200 的套餐，就不要让它闲着。

## 5. 注意事项
- **流量消耗**：CLI 界面流量极小，非常适合移动网络。
- **安全性**：强烈建议使用 **Tailscale** 等私有网络连接，不要将 SSH 端口直接暴露在公网。
- **输入效率**：手机端建议配置常用的缩写词（Snippets），方便快速下达 `PLAN`, `DO`, `CHECK` 指令。
