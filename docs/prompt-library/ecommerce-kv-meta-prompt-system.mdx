# 电商 KV 视觉系统元提示词 (10-Poster Set)

本系统由沐阳 (@yyyole) 迭代开发，旨在通过一个全能的元提示词，指挥多模态 AI 完成从产品识别到全套电商视觉海报提示词的生成。

---

## 🛠️ 第一步：产品信息智能提取 (Vision Extraction)
*上传图片后，AI 将自动分析：*
- **品牌与类型**：Logo、分类、规格。
- **核心卖点**：包装文案、认证标识、视觉特征。
- **审美基因**：配色方案 (HEX)、字体风格、材质质感。

## 🎨 第二步：风格与排版路由 (Style Routing)
*基于识别结果，AI 自动推荐或供用户选择：*
- **7 大风格**：杂志编辑、水彩艺术、科技未来、复古胶片、极简北欧、霓虹赛博、自然有机。
- **6 大排版效果**：玻璃拟态、3D 浮雕、手写体、网格对齐等。

## 🚀 第三步：生成 10 张海报提示词系统 (Swarm Generation)
*系统将一次性输出：*
1. **海报 01 - 主 KV (Hero Shot)**：严格还原包装与 LOGO。
2. **海报 02 - 生活场景 (Lifestyle)**。
3. **海报 03 - 概念可视化 (Concept)**。
4. **海报 04-07 - 细节特写/评价 (Details/Review)**。
5. **海报 08 - 品牌故事 (Brand Story)**。
6. **海报 09 - 参数表 (Specifications)**。
7. **海报 10 - 使用指南 (Usage Guide)**。

---

## 📝 核心质量准则 (Guardrails)

1. **高度还原**：必须明确指令 Agent 保持包装、颜色、LOGO 位置、文字内容与原始图 100% 一致。
2. **中英双语排版**：所有文字必须提供【中英堆叠】、【中英并列】或【中英分离】三种专业格式。
3. **视觉一致性**：10 张海报共享同一套配色系统与品牌调性。

## 📥 使用方法
1. 将本文件内容复制。
2. 开启支持视觉的 AI 助手（如 Claude 3.5 Sonnet）。
3. 上传你的产品图，发送提示词，开启自动化生成流程。
