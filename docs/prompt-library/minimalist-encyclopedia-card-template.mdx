# 极简百科信息卡片提示词模板 (Encyclopedic Info-Card)

本模板源自沐阳 (@yyyole) 的“芦笋百科”案例，适用于将任何复杂的主体（食材、代码模块、Agent 角色、历史人物等）转化为高审美、高密度的信息卡片。

## 1. 核心提示词框架

### [主体名称] 深度视觉化指令集

**视觉哲学：** 瑞士平面设计、包豪斯功能主义、Dieter Rams 十条准则。

**一、 构图协议 (Composition)**
- **布局**：12列网格，8pt 基准对齐，2:3 竖版。
- **视觉分区**：
  - 顶部 (30%)：主视觉特写 + 极简标题排版。
  - 中部 (60%)：9大核心信息模块（网格化）。
  - 底部 (10%)：溯源数据 + 设计元数据。

**二、 视觉规范 (Visual Identity)**
- **调色盘**：从 [主体] 提取自然主色，配合大地色系（#FAFAFA 背景）。
- **字体系统**：
  - 标题：超细无衬线体 (Helvetica Neue UltraLight)。
  - 正文：现代几何体 (DIN/Futura)。
  - 数据：等宽字体 (Roboto Mono)。
- **图标系统**：2pt 线宽，圆角一致，负空间利用。

**三、 核心信息模块 (Content Modules)**
*(根据主体调整以下 9 个模块内容)*
1. **成分/结构详解** (饼图/条形图数据)。
2. **功效/核心价值** (图标化展示)。
3. **方法/操作矩阵** (参数与星级评价)。
4. **时序/进化指南** (时间轴或圆盘图)。
5. **关联/搭配矩阵** (协同关系图)。
6. **人群/应用画像** (分类图标)。
7. **约束/注意事项** (警告色块)。
8. **选购/质量标准** (多维度判断图示)。
9. **深度知识扩展** (历史与数据可视化)。

---

## 2. 为什么这套模板有效？

1. **降低 AI 的随机性**：通过精确的字体磅值 (48/32/24...) 和色值 (#2C2C2C)，强制 AI 维持高标准的审美。
2. **数据墨水原则 (Data-Ink Ratio)**：遵循 Edward Tufte 的理论，去除非必要装饰，让每一像素都承载信息。
3. **呼吸感 (Breathability)**：通过 25% 的模块间留白和 1.8 倍行间距，确保高密度信息不拥挤。

## 3. 适用场景建议

- **Repo 分析摘要**：用此风格生成一个项目的 `README` 概览图。
- **量化自我报告**：展示 [3.5GB 健康数据](../../blog/2025-12-25-agentic-data-mining-apple-health-3gb.md) 的核心 Insights。
- **Agent 入职合同**：以“数字员工信息卡”的形式展示 Agent 的能力与约束。
