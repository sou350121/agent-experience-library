# 仿生设计全流程展示提示词模板 (Bionic Design Workflow)

本模板源自沐阳 (@yyyole)，适用于展示从自然灵感到工业产品的完整进化过程，非常适合用于设计作品集、提案汇报或创意脑暴。

---

## 📐 构图协议 (The 5-Stage Layout)

**整体布局：** 横向五分区，从左至右排布，比例均为 1/5。用极细灰线或虚线分隔。

### 第一阶段：自然观察 (Nature)
- **视觉**：高清微距摄影。
- **重点**：真实质感、饱满色彩、生态摄影风格。

### 第二阶段：灵感提取 (Extraction)
- **视觉**：科学分析图。
- **重点**：白色标注线 + 中文标签。涵盖：色彩、形态、纹理、功能。

### 第三阶段：概念草图 (Sketch)
- **视觉**：手绘线稿/铅笔素描。
- **重点**：标注尺寸、材质说明，保留思考的痕迹感。

### 第四阶段：产品渲染 (Rendering)
- **视觉**：3D 工业渲染。
- **重点**：纯白/渐变背景，精确的光影与工业材质表现。

### 第五阶段：场景应用 (Context)
- **视觉**：高级商业摄影。
- **重点**：精致布景、专业摄影棚光效、时尚杂志质感。

---

## 📝 核心提示词指令 (Instructions)

### [顶部标题系统]
> 顶部中央添加大标题："[主体名称] 仿生设计"。
> 底部标注流程：自然观察 → 灵感提取 → 概念草图 → 产品渲染 → 场景应用。

### [视觉规范]
- **背景**：纯白或 #FAFAFA 浅灰。
- **字体**：中文黑体/宋体，极简精致。
- **排版风格**：专业设计作品集 (Portfolio) 逻辑。

## 🛠️ 实战演练建议 (万物皆可仿生)

1. **选题建议**：
   - **鸡头** → 云台/稳定器 (Gimbal)。
   - **荷叶** → 超疏水涂层/纳米服饰。
   - **翠鸟** → 高铁车头。
   - **莲蓬** → 分布式传感器/蜂窝结构音响。
2. **多 Agent 协作**：
   - 使用 **Claude Code** 帮你分析某种生物的机械特性。
   - 使用 **Midjourney** 结合此提示词生成视觉全案。
   - 使用 **[Qoder](../../docs/tools/qoder-repo-wiki-deep-dive.mdx)** 来记录这个设计的架构逻辑。
