# 07 闭环：小白如何赚到第一美金

在 Vibe Coding 时代，写代码已经不再是最高的门槛。**如何把你的代码变成钱，并且稳稳地收进兜里**，是你要学习的最后一课。

## 1. 独立开发的“定价玄学”

很多小白会有个误区：“我写的东西很简单，是不是应该便宜点？”

**错误！**

```text
    定价与精力分配 (Pricing vs. Effort)
    
    [ 低定价模式 ]                [ 高定价模式 ]
    
    维护成本                      服务价值
     ^                            ^
     |  !!! [海量低质用户]         |
     |      |                     |   +----------+ [核心高质量客户]
     |      |                     |   |          |
     |      |                     |   |          |
     +------------> 收入          +------------> 收入
    (每天回邮件，没空写代码)         (收贵点，服务好那批懒人)
```

**心法：** 宁可定价 $19/月 服务 100 个高质量用户，也不要 $1/月 服务 1900 个挑剔用户。

## 2. 收款工具：从个人到公司

你不需要一开始就去注册一家美国公司，这太重了。

| 阶段 | 方案 | 为什么？ |
| :--- | :--- | :--- |
| **0 - 10 客户** | **个人直接收款** | 使用 [Creem.io](https://creem.io) 这种免公司主体的套壳服务，快速测试。 |
| **10 - 100 客户** | **Stripe Atlas** | 注册美国怀俄明州 LLC。正式、合规，且第一年有手续费减免。 |
| **银行账户** | **Mercury** | 纯线上银行，小白友好，免管理费。 |

## 3. 防御性运营：把“证据”写进代码里

海外收款最怕的是 **Dispute（争议/退款）**。Stripe 往往更偏向买家，为了保护自己，你要让你的 Agent 在开发时就加入以下“证据自动化”：

1.  **自动欢迎邮件**：证明用户已成功注册并收到说明。
2.  **续费提醒 (7/3/1 原则)**：在自动扣费前 7天、3天、1天分别发邮件，防止用户以“我不记得订过”为由投诉。
3.  **使用记录日志**：当用户质疑没使用服务时，这些日志是你胜诉的铁证。

## 4. 最小练习（10 分钟）

请思考并让 AI 协助你：
1.  **定义定价**：为你之前练习的产品设计一个“高价值、针对懒人”的订阅方案。
2.  **规划邮件**：让 AI 写一份“防止退款争议”的续费提醒邮件模板。
