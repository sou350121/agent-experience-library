# 空间智能：超越语言的下一个前沿

## 1. 为什么语言模型 (LLM) 还不够？

目前的 LLM（如 ChatGPT, Claude）是基于文本的。虽然它们能写代码、做推理，但它们对**物理世界的三维空间、物体属性和动态交互**缺乏本质理解。

- **局限**：2D 图像/视频生成虽然逼真，但在 3D 结构上往往经不起推敲（指纹不对、光影不连续、物理规则违背）。

## 2. 空间智能 (Spatial Intelligence) 的定义

李飞飞提出的空间智能，是指 AI 理解并模拟 3D 物理世界的能力：
- **深度感 (Depth)**：物体之间的前后距离。
- **物理性 (Physics)**：重力、遮挡、碰撞。
- **动态交互**：人在空间中走动、拿取物体的逻辑。

## 3. World Labs 的愿景：创造力的民主化

李飞飞的创业项目 **World Labs** 旨在打造“世界模型”，让创造 3D 世界变得像写字一样简单。

### 3.1 降低门槛
- **现状**：创造 3D 场景需要学习 Blender, Unity 等专业工具，门槛极高。
- **未来**：AI 充当“神笔”，普通人可以通过自然语言或简单引导生成完整的 3D 交互空间。

### 3.2 赋能创造者 (Empowering Creators)
这不是为了取代设计师，而是让有创意但不懂复杂工具的人，能够具象化脑中的想像力。这是人类创造力的**民主化**。

## 4. 结论

如果我们说 LLM 模拟了人类的“语言思维”，那么空间智能就是在模拟人类的“具身认知”。它是 AI 进入物理世界（机器人、AR/VR、智能制造）的关键拼图。
