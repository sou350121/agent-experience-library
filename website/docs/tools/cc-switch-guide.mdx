# 工具指南：使用 CC Switch 管理你的 Agent 军团

**CC Switch** 是一款专为 AI CLI 工具设计的可视化配置管理应用。它支持 **Claude Code**, **Codex** 和 **Gemini CLI**，让你彻底摆脱手动编辑 `config.toml` 的烦恼。

## 1. 核心功能

- **多供应商支持**：预设了 Claude 官方、DeepSeek、智谱 GLM、Kimi K2、Mini Max 等 17 个途径。
- **MCP 管理**：可视化添加和开关 MCP 服务，无需手动修改 JSON。
- **Skills 管理 (Claude 专属)**：一键扫描并安装 Hugging Face Skills 或自定义技能。
- **云同步**：支持通过主流云盘同步配置，实现多设备一致性。

## 2. 安装方法

### Windows / Linux
前往 [GitHub Releases](https://github.com/farion1231/cc-switch) 下载对应版本的安装包。

### macOS
推荐使用 Homebrew 安装：
```bash
brew tap farion1231/ccswitch
brew install --cask cc-switch
```

## 3. 实战技巧：配置 Codex 接入 GPT-5.2

以使用中转站接入最新的 `gpt-5.2` 为例：

1. **选中 Codex**，点击右上角橙色 `+` 号。
2. **API Key**：填入你的令牌。
3. **API 地址**：填入中转站地址（如 `https://open.cherryin.ai/v1`）。
4. **编辑配置**：在下方的 `config.toml` 编辑框中，将 `model` 修改为 `openai/gpt-5.2`。
5. **推理参数**：将 `model_reasoning_effort` 设置为 `low` 或 `medium`（平衡速度与成本）。
6. **启用**：点击保存并启用，重启 CLI 即可生效。

## 4. MCP 与 Skills 管理

### MCP 管理
在 CC Switch 的 MCP 窗口中，你可以直接点击 `+` 号添加 MCP 链接。例如添加 `context` MCP，添加后即可在 Codex 或 Claude Code 中直接调用，无需重启或手动改配置。

### Skills 管理
选中 Claude 工具后，进入 Skills 窗口：
1. 配置仓库连接（如 `https://github.com/huggingface/skills.git`）。
2. 点击刷新，扫描可用技能。
3. 一键安装，快速赋予 Claude 处理模型训练、数据分析等能力。
