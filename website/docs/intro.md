# Agent 经验库 入门指南

欢迎来到 **Agent 经验库**！这是一个专门记录 Agent 使用心得、工具对比、Prompt 技巧与实践知识库。

## 库的内容结构

这个库主要分为以下几个核心板块：

- **[工具链](./tools/intro)**：Cursor、Claude Code 以及各类 IDE/CLI Agent 的使用方法。
- **[计划与范式](./planning/intro)**：如何让 Agent 更好地规划任务（强计划 vs 弱计划）。
- **[模型对比](./model-comparisons/intro)**：不同模型在 Agent 场景下的实测对比（如 GPT-5.2 vs Claude 4.5）。
- **[能力边界](./capabilities/intro)**：Agent 能做什么，不能做什么，以及常见的失败模式。
- **[Prompt 库](./prompt-library/intro)**：高质量的 Prompt 集合，涵盖 UI/UX、代码、规划等场景。
- **[案例复盘](./case-studies/intro)**：真实的实战案例，记录决策过程与反思。

## 如何使用本库

1. **查阅文档**：通过左侧侧边栏按类别查阅。
2. **追踪动态**：查看 [日更动态](/blog) 了解最新的 X 平台资讯提炼。
3. **参与贡献**：如果你有好的经验，欢迎提交 PR。

## 核心理念

我们追求：**可检索、可复用、可复现**。每一篇对比或案例都力求提供充足的证据和可操作的建议。
