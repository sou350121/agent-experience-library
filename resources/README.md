## 🧠 核心洞见

1️⃣ **情报源的“值函数”过滤**
- **逻辑思考**：信息通胀时代，获取信息不难，难的是如何建立过滤机制。
- **启发**：关注那些具有“Taste”的人（见影响力图谱），本质上是在借用他们的“决策值函数”来为自己筛选高质量信号。

2️⃣ **社区是 Agent 的“社会化学习场”**
- **逻辑思考**：Agent 的能力不仅来自数据，也来自人類在社区中碰撞出的“非共识”。
- **启发**：不要只读文档，要去 GitHub Issue 和 Discord 频道里看大家踩过的坑，那是文档里没有的真实边界。
