宝玉老师（@dotey）最近分享了一个非常典型的争议：**既然大模型已经这么强了，直接描述问题不就行吗？为什么还要学 Prompt？**

### 核心观点提炼

1. **简单任务 vs 复杂任务**：
   - 问天气、查单词确实不需要 Prompt。
   - 复杂任务需要 Prompt 作为**“数学公式”**。它填补了“指令”与“意图”之间的鸿沟。

2. **从“指令”到“工作流经理”**：
   - 高阶 Prompt 不只是提问，而是串联任务（如：生成大纲 -> 自动写配图指令）。
   - 它让 AI 从聊天机器人变成**“自动化生产线”**。

3. **稳定性与工具化**：
   - 聊天是一次性的，而专业的 Prompt 是**“程序”和“软件”**。
   - 好的 Prompt 是经过无数次试验优化的“秘密配方”，能实现稳定高质量的产出。

4. **角色转变**：
   - Prompt 让用户从“提问者”变成了**“指挥 AI 干活的工程师”**。

### 典型案例
- **学术转科普**：设定读者画像与比喻风格，避免术语堆砌。
- **YouTube 字幕提取**：自动标注发言人、分章节，而非纯文本。
- **信息图生成**：将文章直接转化为专业视觉内容。
