今日必读：潘致雄老师关于 AI Agent 底层逻辑的分享。

### 核心金句
- **定义**：Agent 是 LLM 的“外骨骼”，给大脑装上了手脚。
- **思维转换**：构建 Agent 需要的是“管理思维”而非“编程思维”。你是在招募并授权一个“数字实习生”。
- **架构原则**：切忌“大而全”，要像控制人力成本一样控制 Token 成本和工具调用预算。

<!-- truncate -->

### 实践启示
不要试图教 AI 怎么写代码，要教它怎么用工具。通过 System Prompt 为它预设“操作手册”，让它在自主决策中完成 MVP 闭环。
