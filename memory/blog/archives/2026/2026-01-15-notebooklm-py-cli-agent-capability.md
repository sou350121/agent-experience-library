# 把 NotebookLM 搬進命令行：從「產品」到「Agent 能力組件」的飛躍

最近 GitHub 上冒頭的神器 **notebooklm-py**，讓 Google NotebookLM 的超強 RAG 與播客生成能力，通過 CLI 和 Python API 徹底對開發者和 Agent 開放了。

### 1️⃣ 為什麼這件事非常酷？
對於真正的 Power User 和 Agent 開發者來說，網頁版的點選是效率的枷鎖。
- **可編程性**：現在你可以寫一個腳本，自動把 50 篇 PDF 論文餵進 NotebookLM，然後直接拿到總結。
- **Agent Skill**：它可以作為 Claude Code 的一個 Skill，讓你的 AI 助手在寫代碼時，隨時調用 NotebookLM 的研究能力。

### 2️⃣ 核心功能拆解
- **自動化管道**：支持 URL、PDF、YTB 視頻、Google Drive 的批量導入。
- **多模態輸出**：一行命令生成音頻播客、思維導圖、甚至測驗和學習指南。
- **三位一體用法**：Claude Skill（自然語言調用）、CLI（終端操作）、Python API（系統集成）。

### 3️⃣ 對 Agent 的啟示
未來的知識管理不是「手動存檔」，而是「自動流轉」。`notebooklm-py` 是構建「信息輸入 -> AI 提取 -> 多維輸出」這條自動化管道的關鍵積木。
