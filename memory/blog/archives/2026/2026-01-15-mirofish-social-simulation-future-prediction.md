# MiroFish：基于多智能体的平行世界预测引擎

如果你能创建一个现实世界的「平行数字世界」，并动态观察不同决策下的未来走向，你会怎么做？

**MiroFish** 正在尝试将大模型 Agent 从单纯的助手进化为社会群体，通过模拟社会演化来辅助现实预测。

### 1️⃣ 什么是 MiroFish？
MiroFish 是一个基于多智能体技术的预测引擎。它通过上传现实世界的种子材料（如报告、突发事件描述），自动构建一个包含成千上万个拥有独立人格、记忆和逻辑的 AI 智能体的平行空间。

### 2️⃣ 核心技术亮点
- **增强型时序 GraphRAG**：精准还原事件背景，处理具有先后顺序的复杂记忆。
- **高保真人格模拟**：自动为 Agent 设定详细属性、背景及社交媒体活跃习惯。
- **上帝视角干预**：用户可以动态注入变量（如改变政策、注入流言），观察社会情绪的演变。

### 3️⃣ 典型应用场景
- **决策预演**：在零风险环境下模拟政策或公关危机的后续发展。
- **社会洞察**：通过与特定 Agent 对话或开展问卷调查，深度挖掘群体心理。
- **创作辅助推演**：如小说结局推演、脑洞设想（“权力的游戏结局会如何？”）。
