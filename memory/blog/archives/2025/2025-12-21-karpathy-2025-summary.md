AI 巨擘 Andrej Karpathy 发布的 2025 年终总结，勾勒出了大模型从“聊天框”走向“逻辑幽灵”的剧变轨迹。

### 核心观点提炼
1. **RLVR 统治训练**：算力不再只堆在预训练，而是转向“可验证奖励的强化学习”。这是 O1/O3 能力质变的核心。
2. **锯齿状智能 (Jagged Intelligence)**：AI 并非全才，而是在可验证领域极强、在模糊领域极弱的“幽灵”。
3. **本地 Agent 的胜利**：Claude Code 证明了“住在电脑里”的本地代理比云端容器更具交互优势。
4. **Vibe Coding 门槛跨越**：2025 是人类可以只用英语构建复杂程序的转折点，代码变得“随用随弃”。
5. **LLM 的 GUI 进化**：从命令行（聊天）转向空间化、视觉化的交互（Nano Banana 范式）。

### 给开发者的启示
2025 年的进步并非来自模型规模，而是来自**推理时计算 (Test-time Scaling)** 和 **RL 训练**。理解这一点，你就能明白为什么 Vibe Coding 配合强大的推理模型（如 GPT-5.2）会产生如此恐怖的生产力。
