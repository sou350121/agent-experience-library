今日神操作：Olivia (@0xOliviaPp) 分享了一种“极致剥削” AI 订阅费的玩法，被潘致雄老师称为「智力套利」。

### 核心逻辑
- **智力套利 (Intelligence Arbitrage)**：支付固定订阅费（如 Claude Pro $200 档位），通过高频调用极高成本的顶尖模型，实现价值的不对等收割。
- **移动端“剥削”工作流**：
    1.  **设备**：家里/办公室电脑保持开启。
    2.  **工具**：电脑运行 Claude Code CLI 或类似 Agent。
    3.  **连接**：手机安装 Termius 等 SSH 客户端。
    4.  **场景**：出门在外，用手机远程指挥 Agent 疯狂干活，让 Token 燃烧不停。

<!-- truncate -->

### 为什么这很重要？
这打破了“办公地点”的限制。当 Agent 足够强大到可以自主处理复杂任务时，你的角色从“操作员”变成了“指挥官”。即便在通勤路上，通过简单的手机指令，你也能驱动昂贵的计算资源完成原本需要坐在工位上几小时的任务。
